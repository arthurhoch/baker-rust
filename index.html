baker-rust packages

Based on the original Python BakerCM (PyPI). We ran into installation issues with the PyPI package in some environments, so baker-rust reimplements the same behavior as a single self-contained binary to simplify installs anywhere.

What this is: a Rust reimplementation of BakerCM that ships as a single binary, with APK repositories for Alpine and Wolfi ready for direct install.

Why: the original PyPI package had install issues in some environments; this binary keeps the same recipe/template semantics and secret format, but is easier to deploy.

Alpine repo root: alpine/baker-rust/x86_64/

Wolfi repo root: wolfi/baker-rust/x86_64/

Project source (GitHub): https://github.com/arthurhoch/baker-rust

  • Alpine: APKINDEX.tar.gz | packager.rsa.pub
  • Wolfi: APKINDEX.tar.gz | melange.rsa.pub

Use in Alpine

echo "https://arthurhoch.github.io/baker-rust/alpine/baker-rust" | sudo tee -a /etc/apk/repositories
sudo wget -O /etc/apk/keys/packager.rsa.pub https://arthurhoch.github.io/baker-rust/alpine/baker-rust/x86_64/packager.rsa.pub
sudo apk update
sudo apk add baker-rust

Use in Wolfi (wolfi-dev)

echo "https://arthurhoch.github.io/baker-rust/wolfi/baker-rust" | sudo tee -a /etc/apk/repositories
sudo wget -O /etc/apk/keys/melange.rsa.pub https://arthurhoch.github.io/baker-rust/wolfi/baker-rust/x86_64/melange.rsa.pub
sudo apk update
sudo apk add baker-rust

Note: the repo path includes the architecture subfolder. Directory listings under alpine/baker-rust/x86_64/ and wolfi/baker-rust/x86_64/ serve index pages and keys per architecture.
